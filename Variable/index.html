声明一个变量的4种方式

  • a = 1
  • var a = 1
  • function a(){}
  • let a = 1
  • const a = 1
  • import {a} from "./xx.js"
  • class

let

1. let的作用域在最近的 {} 之间

2. 不存在变量提升，所声明的变量一定要在声明后使用，否则报错

3. let 有暂时性死区(Temp Dead Zone)在代码块内，使用let命令声明变量之前，该变量都是不可用的

4. 不能重复声明,重复let a会报错

const

1. const的作用域在最近的 {} 之间

2. 不存在变量提升，所声明的变量一定要在声明后使用，否则报错

3. const 有临时死区(Temp Dead Zone)在代码块内，使用const命令声明变量之前，该变量都是不可用的

4. 重复const a会报错，不能重复声明

5. 只有1次赋值机会，而且必须在声明的时候立马赋值
