对象

1.创建对象的3种方法

  • new Object()
  • Object.create(null)
  • {}字面量
    • Object.create(null)会创建一个真正的空对象，没有__proto__属性。第一个参数是新创建对象的原型对象

2.对象属性缩写

3.动态属性名

4.方法定义

对象属性也可以是一个函数、getter、setter方法

5.扩展属性

深拷贝一个对象
  • for循环，将obj的key和value赋值给_obj的key和value
  • _obj = Object.assign({},obj)
  • _obj = {...obj}

6.变更原型

    将obj1的原型设置为obj
  • var obj1 = Object.create(obj)
  • Object.getprototypeof(obj)获取一个对象的原型

7.Object.defineProperty(obj,prop,descriptor)

在一个对象上定义一个新属性，或者修改一个对象的现有属性，并返回这个对象

8.null与undefined

undefined是window的只读属性

Number(undefined) 返回NaN

Number(null) == 0

9.设置一个属性为只读

  • 使用对象的get方法
  • 使用Object.defineProperty()的writable属性，设置为false

10.设置一个对象的属性为不可遍历

使用Object.defineProperty()的enumerable属性，设置为false(默认就是false)

10.Object.getOwnPropertyDescriptor(obj,prop)

返回指定对象上一个自有属性的描述对象

11.Object.defineProperties(obj,props)

在一个对象上定义或修改多个属性，并返回该对象

12.遍历对象的属性

  • Object.keys()返回对象自身的属性(而不是继承的)，只返回可枚举属性
  • Object.getOwnPropertyNames()返回对象自身的属性(而不是继承的)，还返回不可枚举属性
